# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/4d220bda-1228-44ee-b2ca-5d5e9dc43072

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the Express app:
   `npm run dev`

The React app is served through Express in development, with Vite attached as middleware. API routes live under `/api`.
